# Bounce Rogue / 弹球肉鸽 —— 设计文档 v2

> **v2（2026-07-18）**：依市场调研（见 `RESEARCH.md`，含全部出处）整体重定骨架。v1 的「实时 paddle 抄 Roguebreaker」被证伪——Roguebreaker 八年 25 条评测≈零市场，移动端实时 paddle 无成功近例（手指遮挡球路 + 触屏无法精确停位，结构性缺陷）。v2 =
> **竖屏回合制瞄准发射（Ballz/Holedown 形态）× BALL x PIT 合球化学 × Balatro 规则卡 × 可追责 RNG**。
> 可爱收集/图鉴/成就/无永久数值沿 v1 不变。root `CLAUDE.md`（引擎契约、部署铁律、git 纪律、iOS 流水线）先读，本文件只讲本游戏专属设计。

## 0. 一局是什么样（一句话）

拖动瞄准、松手把一队**元素小精灵球**射向上方砖阵看它们疯弹，砖阵每回合下压一行；碎砖攒能量二选一**规则卡**，捡球蛋合成进化球种；砖压过底线扣血、血尽 run 终。**15~20 分钟一局，三世界 + Boss，通关解锁更高难度层。**

## 1. 为什么是这个形态（调研裁决，细节见 RESEARCH.md）

- 移动端唯一被三层市场（Ballz 超休闲 / Holedown 精品买断 / Peglin 肉鸽）共同验证的操作 = **竖屏单手回合制 aim-shoot**；实时 paddle 手机上零成功案例。
- 品类 50 年公认教训：接球 miss 是负担不是乐趣。所有现代成功者都在**把「接球」删掉，乐趣全押「出手那一发」**（holedown 回合制 / BALL x PIT 自动连发）。
- **RNG 控制感是口碑生死线**（Peglin/Ballionaire 差评同款：「没有一个决策真正重要」）。回合制弹道是确定性的 → 瞄准预览 + 全种子化 = 宁可玩家骂自己手歪，不让玩家骂骰子。
- 与头部 BALL x PIT 移动版（$9.99 买断、实时反应）**双错位**：回合制可深思 + 免费广告制 + 可爱收集。免费+广告端目前没有真 build 深度的弹球肉鸽——这是我们的空位。
- 副产品：确定性回合物理 = **蒙特卡洛模拟直接可用**（本仓数值文化），auto-player 可做角度搜索，可测性比实时 paddle 好一个量级。

## 2. 核心循环

1. 盘面 **7 列 × 9 可见行**方格砖阵（砖带小圆角，角部反弹按角法线偏转——holedown 的「口袋乱弹」爽点来源）。
2. 一回合 = **拖动瞄准**（虚线预览前 2 段反弹）→ 松手**全队连发**（每颗间隔 ~80ms）→ 弹跳结算 → 球落回底部 → **砖阵整体下压一行** + 顶部按关卡脚本生成新行。
3. 碎砖攒能量槽，满 → 回合间弹**二选一规则卡**（带重抽 / banish；看激励视频出第三张）。
4. 掉落物：**球蛋**（孵新球）/ 宝石（治疗/能量）。球碰到即收，回合结束统一结算。
5. 砖被压过底线：对玩家造成其**剩余 HP 的伤害（单块上限 30）**后消失。**玩家 HP 100，血尽 = run 结束**——唯一失败条件；渐进失败，无瞬死（学 mines「一条干净规则」+ 调研「失败必须可预见」）。
6. 每世界 ~20 回合 + **Boss 砖**；Boss 死 → 世界通、大奖励；**三世界通关 = 胜利**。通关解锁难度层 D2~D9（砖 HP / 下压节奏上调，opt-in，Peglin Cruciball 式）。

**回合节奏防呆**（P1 必备三件套）：回合内球速随时间渐加速（1→1.5×）+「召回」键 + 连续 40 次反弹未命中任何砖 → 弹道注入 0.5° 偏转（Ballz 水平死循环是品类知名 bug）。

## 3. 球种化学（主轴 = build 深度 + 收集钩子，borrow BALL x PIT）

- 球 = **元素小精灵**——可爱收集主题与合成系统天然咬合：火+水=蒸汽，**配方能被常识猜中，猜中本身是奖励**（调研：BALL x PIT 被点名最多的爽点是「可推理的化学」）。
- 初始 3 颗 L1 普通球；球蛋孵出新球（种子随机）。**同种两颗合级** L1→L2→L3；**两颗 L3 有配方 → 进化**成新种（配方进图鉴）；**无配方 → 融合**（叠双方效果，外观拼合）。进化球还能再融合 = 两层组合空间（比 VS 式单层进化耐玩的原因）。
- **种类上限 4、总球数上限 12**——逼取舍凑配方；超上限时必须**吞并**（牺牲一颗给另一颗喂经验）。
- 首版 ~10 基础种 + ~12 进化配方。基础例：火（碎砖爆炸）、电（连锁跳 3 砖）、穿透、分裂、毒（持续掉血）、冰（冻结下压一回合）、幽灵（首下穿砖）、磁（吸掉落物）、光（暴击）、石（高基伤低速）。进化例：火+电=等离子（爆炸+连锁）、冰+穿透=冰锥（穿冻）、火+光=太阳（灼烧全行）…
- **伤害 = 球基伤（按等级）× 卡的加法区 × 卡的乘法区**（Balatro：加法开局、乘法收尾）。

## 4. 规则卡（副轴，首版 20 张精品池，拒绝 +10% 数值卡）

每张都**改一条规则**，普通/稀有/史诗。样例：

- **底线弹床**（史诗）：砖首次触底不扣血、被弹回两行
- **三思**：瞄准预览延长到 4 段反弹
- **静止时间**（双刃）：下压改每 2 回合一次，但砖 HP +40%
- **头槌**：每回合首颗球伤害 ×3
- **口袋玩家**：球在砖缝连续反弹 ≥6 次 → 该回合伤害 ×2
- **全清红利**：一回合清空全场 → 跳过 2 次下压 + 掉双蛋（强 build 的**提前结算手段**，Balatro skip-blind 等价物）
- 蛋壳暴击 / 连锁保险 / 毒雾扩散 / 底线商人 /…（P2 扩到 ~40）

## 5. 砖块生态（~8 种 + Boss，可图鉴，学 mines 怪物表）

普通 · 硬砖(高 HP) · 爆炸砖(死时清邻) · 蛋砖(必掉球蛋) · 硬石(不可碎，只被爆炸/穿透清——挡弹道做地形) · 治疗砖(碎了回血) · 冻砖(碎了停一次下压) · 诅咒砖(压线伤害翻倍，逼优先清)。
**Boss 砖**：2×2 起步、大血 + 行为（召唤小砖 / 自身前冲一行 / 护盾轮转），每世界一个，死亡 = 满屏结算高潮 + 专属掉落。

## 6. RNG 纪律（生死线）

- **弹道零随机**：给定瞄准角，轨迹完全确定（预览即真相）。随机只存在于：砖行生成、球蛋孵化、卡池抽取、掉落——**全走 mulberry32 种子，局内可查种子**（与 blockblast「种子可查」同一品牌语言）。
- **保底**：蛋砖 X 回合内必出；治疗有下限供给；banish 后卡池收敛（数值全靠 `tools/sim` 校准，不拍脑袋）。

## 7. Juice 手感（四件套，克制分级——Juice It or Lose It + holedown 实践）

碰撞挤压（球压扁、砖弹起）· 击碎粒子 + **分级**屏震（连锁才大震）· **连击音高按音阶递增**（holedown 招牌）· FX 只标记大事（合球/进化/Boss/全清）。精灵有脸、待机眨眼（juice talk 原教旨：给球加眼睛）。美术先 emoji/纯色占位、手感验证后走本地 Flux（`game-art-pipeline`）。

## 8. 元进度（broader than deep，**无永久数值**——v1 立场，调研强化）

图鉴（球种/配方/砖/卡，DOM 浮层）+ 里程碑解锁新球种新卡**进池**（横向）+ 皮肤 + **~120 成就**（数据表驱动 + 运行时断言）+ 难度层。**只存**：图鉴/解锁/成就/最高分/皮肤/当局快照。

## 9. 广告钩子（引擎 `Ads`，自备 adUnits，绝不复用他游戏）

- **激励-复活**：血尽 → 看广告半血续（每 run 1 次）。
- **激励-第三张卡 / 重抽**：二选一时。
- **插屏**：世界间隙（不打断回合内）。

## 10. 工程（严守 `engine/README.md`）

- 加载顺（= 依赖顺）：引擎 → `prng` → `bricks` → `balls`（球种/配方数据）→ `cards` → `core`（回合状态机/弹道模拟/下压/HP/掉落）→ `codex` → `storage` → `achievements` → `themes` → `render` → `main`。纯逻辑层双导出、node 可单测。
- **core 的弹道模拟必须是纯函数**：`simulateTurn(state, angle) → { newState, events[] }`。渲染层按事件回放动画，逻辑与表现彻底分离——**预览线、蒙特卡洛、auto-player 三件事共用这一个地基**，P1 第一优先级。
- 游戏时钟非墙钟（回放动画/FX 走 `gameMs`）；**事件流 `s.events`**（brickBreak/combo/eggDrop/merge/evolve/fuse/lineHit/bossPhase/perfectClear/cardOffer/death…）供成就与音效统一消费。
- 存档（单 JSON blob）：元进度 + 当局快照（砖网格/球队含等级/已选卡/HP/回合号/世界号/种子）。**SAVE_VERSION 纪律**、**`?v=N` 纪律**照 root CLAUDE.md。开放 map 字段保持空默认（snake 实踩 Critical）。
- **零硬编码文案**：`T('key')` + 嵌套 locales，en 基准 + zh-CN 首发，`check-locales` 0 fail，非定长文案过 `wrapLines`。
- iOS 壳：Cap6 三件套（package.json / capacitor.config.json / resources）+ codemagic workflow 4 vars，照抄 minesweeper。

## 11. 红线（v2 修订）

- **合球连锁结算**：进化/融合在回合结束统一结算、迭代到稳定、**上限 8 轮防死循环**，单测覆盖。（v1 红线保留）
- **回合时长失控**：球多+反弹多会把回合拖长（holedown 实测教训，为此专门加机制压短）——渐加速/召回/死循环偏转三件套 **P1 就要有**，sim 断言 P95 回合模拟时长 < 25s（游戏时间）。
- **跨引擎浮点**：node sim 与 WebKit 真机**不要求逐位一致**；断言用不变量（总伤害/砖数/HP 范围/球数≤上限），不逐位比轨迹。种子回放只在同引擎内保证。
- v1 的「下降砖 vs 连续球穿模」红线**自然消失**——下压发生在回合间、场上无球（回合制的隐藏红利）。
- ⚠ **措辞死线**（学 solitaire）：预览线只画前 N 段，宣传语绝不能说「完全可预测」；长链弹跳仍是惊喜源——这是特性不是 bug。

## 12. 分阶段实施（先把核心循环做好玩，别堆系统）

1. **P1 核爽**：回合状态机 + 确定性弹道 sim（纯函数）+ 下压/HP/掉落 + 3 球种 + 攒槽二选一（5 卡）+ 预览线 + 节奏防呆三件套 + node 单测 + `tools/sim.js` 蒙特卡洛（校准砖 HP 曲线/伤害曲线/HP 经济）→ **先确认「瞄准-齐射-口袋乱弹-压力线」手感成立**再往下。
2. **P2 build**：球种 10 + 配方 12 + 合级/进化/融合/吞并 + 卡 20 + 砖 8 + Boss 3 + juice 四件套 + 音效。
3. **P3 元层**：图鉴 / 里程碑 / 皮肤 / ~120 成就 / 难度层 / 存档续玩。
4. **P4 变现上架**：Ads + i18n 补语言 + iOS 壳 + 商店素材（名字避 ballz/holedown 字样，防 4.3 clone-spam）。
