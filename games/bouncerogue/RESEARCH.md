# 报告一：Steam/PC 端产品全景

# Steam「弹球/打砖块 × Roguelike」品类调研报告（截至 2026-07）

## 1. BALL x PIT —— 品类现任头部（重点）

**基本盘**
- 开发/发行：Kenny Sun（个人开发者 +「& Friends」小团队）/ Devolver Digital；2025-10-15 发售，PC/PS5/Xbox Series/Switch
- 定价：$14.99
- 销量：**发售 5 天 30 万份，2025 年 12 月破 100 万份**；2026 年三个免费大更新（1 月 Regal / 4 月 Shadow / 7 月 Naturalist）
- 评价：Steam **14,700+ 条评测，95% 好评（Overwhelmingly Positive）**；OpenCritic 92%

**核心循环**：Breakout 式弹球下坑打怪（自动连发、边走边打）+ 局内三球合成进化 + 局外城建基地产资源、解锁新角色/新球。媒体定位「**Balatro meets Vampire Survivors**」。

**玩家公认爽点**（按 VaporLens 评测聚类提及量排序）：
1. **上瘾的循环本身**（99 次提及）——「satisfying, replayable runs」
2. **合球进化/融合**（32 次）——「Bomb + Poison = Nuke」这类可枚举、可炫耀的组合是被点名最多的独创机制
3. **城建 meta**（43 次）——局外长线目标
4. 美术/配乐「简洁而有效」+ 整体打磨

**公认痛点**：
- **后期重复**（76 次提及，负面榜第一）：6-40 小时后花样见底；每关要用多个角色重打才能推进
- **刷子感/强制 grind**（69 次）：成就与 DPS 检查点像硬性门槛
- **build 同质化**（23 次）：RNG + DPS check 收窄可行流派
- **性能**：后期满屏弹球接近 sensory overload；有卡顿/泄漏/崩溃报告
- 前期节奏偏慢（29 次）

**开发者复盘**：灵感直接来自手游 **Punball**（Habby 系 breakout×rogue，「roguelike 版 holedown」）——头部产品是「把手游验证过的循环搬上 PC 并做深」。角色/职业系统一鱼两吃（玩法变化 + 城建进度奖励）。

## 2. Peglin —— 品类老二（Peggle × Slay the Spire）

- EA 2022-04 → 1.0 2024-08；2026-03 v2.0 加第四幕
- $19.99；**约 70 万份、营收约 $6.3M**；Steam ~1.6 万评测，~88%（Very Positive）
- 循环：StS 地图/遗物框架，「出牌」换成打一发弹珠，珠撞钉结算伤害
- 爽点：铺好暴击钉+刷新药水后一发清屏
- **公认痛点**：RNG 压过技巧（「常出现看起来根本赢不了的局」）；平衡失衡（「不抽到少数几个强力珠基本没戏」）；极端评价「你玩的时候没有一个决策真正重要」

## 3. Ballionaire —— pachinko「自动爽机」路线

- 2024-12；~2,200 评测，85%；Balatro 作者 localthunk 公开推荐
- 循环：每回合丢球进 pachinko 板自动结算；玩家全部博弈在 **trigger 摆放规划**，弹跳交给物理（「autobonker」）
- 爽点：「搭好一块财源滚滚的板子的感觉 amazing」
- 痛点：feast-or-famine、「RNG hell」「可行 build 太少」

## 4. Roguebreaker —— 长尾样本（几乎没人玩）

- 2018；约 $1；**仅 25 条评测（76%）**——量级=彻底小众
- **说明单纯「打砖块+过关选升级」这种最直白的拼法在 Steam 上没有市场拉力**，缺合成/进化类雪球机制

## 5. Roundguard —— 早期尝试的教训（2020）

- Peggle + 地牢爬行；446 评测 80%
- 不足：内容太薄（「两小时见完所有东西」）；**meta 进度几乎不留存，「杀死了再来一局的动力」**——正是 BALL x PIT 用城建解决掉的问题

## 6. 其他 2024-2026 新品（均为长尾）

Rogue Patterns / BrickLike / Rogue Bricks / Kill the Brickman / PACHINCRO——个位数到两位数评测量级。未发现有声量的「Puzzle Bobble × roguelike」。

## 小结：规模分层与头部胜因

| 层级 | 产品 | 量级 |
|---|---|---|
| 头部 | BALL x PIT | 100 万+ 份、95% 好评 |
| 次头部 | Peglin | ~70 万份、$6M+、88% |
| 腰部 | Ballionaire | ~2,200 评测、85% |
| 小体量 | Roundguard | ~450 评测、80% |
| 长尾 | Roguebreaker 等 | 约等于零 |

**头部为什么赢**：
1. **消灭等待**：downtime 被改成自动连发+边走边打，弹球第一次有了 VS 式持续输出爽感
2. **合成进化 = 收集欲 + 传播素材**：可枚举配方天然适合攻略/短视频传播
3. **双循环留存**：局外城建接住「输一局也有产出」
4. **RNG 控制感是品类生死线**：Peglin/Ballionaire 差评高度一致——「决策不重要 / feast-or-famine」
5. 发行与传播：Devolver + Twitch 互动；玩法先经手游（Punball）验证

# 报告二：移动端品类形态

## 1. Holedown（grapefrukt / Martin Jonasson）——品类手感天花板

- 付费买断 $3.99，竖屏「向下挖矿」弹球，回合制瞄准发射一串球，砖块带数字血量
- 回合制，单局数分钟；开发者明确说测出「长回合拖沓」后**刻意加机制压短单局**
- 纯买断零广告零内购，「NO ADS」被用户当卖点夸
- **手感核心设计决策**：
  - 整个游戏围绕一个爽点时刻：球钻进砖块背后疯狂乱弹的那一秒，放大成整个游戏
  - **圆角砖块是关键突破**：让球在小口袋里高频野弹（为此修了上百个物理 glitch）
  - **向下射击**：砖块堆叠、打下层有连锁塌落激励；「向上射」被验证「令人难以置信地困惑」
  - ~50 球齐发制造混乱，但 **FX 克制**：只标记真正重要的变化
  - 删掉炸弹类一次性道具（干扰操控简洁性）
  - 公平感：物理真实不暗改，玩家「拥有自己的胜利」
- Juice 教科书：Jonasson & Purho GDC 2012 **「Juice It or Lose It」**（demo 就是打砖块；示例代码 github.com/grapefrukt/juicy-breakout）

## 2. Ballz / Bricks n Balls 系（超休闲）

- 竖屏、划动瞄准→发一串球→砖阵每回合下压一行、到底即死的无尽回合制
- 留存好的原因：规则 3 秒懂 + 每回合一次小决策（找角度打口袋触发连锁）+ 压力线逼近的「one more turn」+ 可随时放下
- 变现：插屏+激励视频为主；Bricks n Balls 至今仍 ~$40 万/月——**该形态生命周期远超一般超休闲**

## 3. BALL x PIT 移动版

- **2026-03-12 上 iOS/Android**，$9.99 买断（try-before-you-buy：免费试玩+单次 IAP 解锁），内容与 PC 同步
- 口碑总体正面但暴露实时形态搬手机的代价：操控 lag、UI 偏小、瞄准圈漂移、部分设备崩溃

## 4. Peglin 移动版

- free-to-start，前 1/3 免费，$8.99 解锁；TouchArcade 4.5/5
- 差评：RNG 过重、触屏瞄准偏难

## 5. 商店里的「打砖块×肉鸽」现状

- Bricks Breaker RPG / B71 / Bounzy! 等——要么超休闲挂 RPG 皮（口碑差），要么爱好者小品
- **BALL x PIT 之前没有真正的头部移动产品**；它是第一个被大规模验证的（先 PC 爆再买断制下移动）

## 6. 单局时长与操作形态共识

- **移动端被验证的主流形态 = 竖屏单手 + 回合制瞄准发射**：超休闲（Ballz）、精品买断（Holedown）、肉鸽（Peglin）三层市场共同验证
- 原因：拖动瞄准精确对应输出角度、无遮挡；回合制适配碎片时间、可随时中断
- **实时 paddle 反弹在手机上没有成功近例**：手指遮挡球路 + 触屏无法精确停位是结构性缺陷（Arkanoid 原作靠旋钮）
- 唯一「实时」成功者 BALL x PIT 也不是 paddle 接球（自动连发不漏接、走位替代 paddle），即便如此移动版仍有操控抱怨

# 报告三：设计方法论

## 1. BALL x PIT 设计复盘

- 合球 = fusion（2 球合成）+ evolution（3 级进化）+ fission，42 配方「化学系统」，乐趣核心在**可推理的发现**（猜「火+光=太阳球」并猜中）；进化后还能再融合 = 多层组合空间（比 VS 单层进化耐玩的原因）
- 自动连发+走位：把「接球生存压力」替换成「弹幕躲避+拾取」，消灭 paddle miss 这一品类最大挫败源；与 VS-like 的差异在**交互性**——纵向通道里策略性打反弹角度
- town 定位「**broader than deep**」（Kenny Sun 原话）：主体横向解锁+薄数值；玩家买账，但也是后期重复感来源之一

## 2. Peglin 的 RNG 挫败结构

- 根因：**决策在投球前、结算在投球后**——瞄准精确但弹跳链混沌，一发歪掉掉一半血；run 逼近 60 分钟放大挫败（一发失误报销一小时）
- 官方解法（实证）：1.0 全面**种子化**（orb 顺序/钉位/Boss 机制）把随机变可追责；但数值补丁治不了「结算 100% 押在混沌物理上」的结构问题
- 教训：**宁可让玩家骂自己手歪，不能让玩家骂骰子**

## 3. Balatro 启发

- ①爽感来自**乘法而非加法**（每 ante ×1.5~2 门槛 + Xmult 叠乘 + 逐级点燃的视听反馈）；②**每个 joker 是一条规则修改**，快乐是「发现系统允许自己作弊」
- 移植边界（AV Club 评 Ballionaire）：「pachinko 一旦球出手就失控——风险和控制你只能留一个」。**弹球肉鸽借 Balatro 必须先回答：物理随机层之上，玩家用什么手段兑现自己的规则修改**

## 4. 传统 paddle breakout 三宗罪与现代改法

- 三宗罪：**尾局追砖**（最后几块砖弹半天）、**miss 惩罚过重**（没接住=掉命，紧张但不有趣）、**进度过缓**
- 现代改法全部指向同一件事：**把「接球」从游戏里删掉或降权，把乐趣重心挪到「打出去的那一发」**
  - holedown：下压式+回合制，miss 掉命 → 压力线逼近（失败渐进、可预见）
  - BALL x PIT：自动连发，瞬间掉命 → 小额掉血

## 5. Juice 手感四件套（弹球类专用）

1. 碰撞瞬间 scale bounce（球压扁、砖弹起）
2. 击碎粒子 + **分级**屏震（普通砖小震、连锁大震）
3. **音高递增连击音效**（holedown 招牌：弹跳音按音阶爬升）
4. 克制原则：弹球开始后画面已经很吵，特效只用来标注「重要的事发生了」

## 6. 元进度之争

- 横向解锁（加内容不加数值）几乎无人反对；永久数值的争议在「被迫用失败换成长」
- 适配原则：**操作技巧占比高 → 适合数值兜底；决策构筑占比高 → 适合纯横向**（保卫每局公平叙事）
- 本仓现稿的「不给永久数值」立场与市场证据一致 ✓

## 7. Run 时长共识

- 甜蜜点 **12~25 分钟**（BALL x PIT 12-20min 被评测明确视为优点）；60 分钟被一致认为过长（Peglin 反面教材）
- 随机品类 run 长度上限 = 玩家能笑着接受「一发毁全局」的时长
- 给强 build 提供主动提前结算手段（Balatro skip blind 等价物）

# 综合：最好玩的配方（四条腿）

> **「holedown 的出手手感 × BALL x PIT 的合成化学与 run 节奏 × Balatro 的乘法计分与规则修改 × Peglin 的反面教材（可追责 RNG + 短 run）」**
> 前人分别验证过每一条，尚未有游戏同时站稳全部四条。

1. **删掉接球，乐趣全押在「出手」上**；失败必须渐进（压力线/掉血），绝不「一次失误掉命」
2. 成长学**可推理化学**而非纯抽卡：配方能被常识猜中，猜中本身是奖励；至少两层组合空间
3. **加法开局、乘法收尾**；每个遗物尽量是规则修改而非 +10%；物理随机之上给玩家确定性兑现手段
4. **RNG 可预算可追责**：种子化 + 保底
5. **run 钉死 15~20 分钟** + 提前结算手段
6. 元进度 **broader than deep**：主体横向解锁，数值只做薄兜底
7. Juice 四件套，克制分级

## 对现 DESIGN.md 的冲击（讨论输入）

- **被证伪**：Roguebreaker 实时 paddle 骨架（Steam 25 评测≈零市场；移动端 paddle 无成功近例、结构性缺陷）；「全漏扣命」的 miss 重惩罚；「分数→球速」的实时难度爬升
- **被强化**：合球系统（头部最大爽点）、图鉴收集、不给永久数值、~120 成就横向解锁
- **可升格**：「下降砖」→ 整面砖阵按回合下压的压力线（Ballz/holedown 同构，天然替代「命」）
- **市场空位**：免费+广告端没有「真 build 深度」的弹球肉鸽（BALL x PIT $9.99 买断、Peglin $8.99、超休闲无深度）；回合制确定性物理天然适合本仓「蒙特卡洛模拟校准」的开发文化
