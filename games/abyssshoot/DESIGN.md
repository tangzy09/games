## 0. 一句话

底部加农炮把小鱼往上射进 5 列,**同数鱼相撞→合并成更大的鱼**(2 条小丑鱼→1 条更大的鱼);射错(不同数)会把那列往下顶,任意列被顶进底部炮台行就输。越合越深,从珊瑚礁一路合到深渊巨兽,**创到新最深的鱼就解锁进图鉴**。

## 1. 核心手感(列式加农炮,整数网格——可测性回到 snake/mines 水平)

- **5 列 × ~9 行**,竖屏锁定(同参考游戏)。列**锚定在顶、往下长、死线在底**。
- 底部炮台:**点哪列就把当前弹药鱼射进哪列**,弹药飞到该列最底空位落定。**只撞列底那一块**(子弹不穿透,埋在里面的够不到)。带**下 N 发预览 + 自由选列**(给 agency,别让避免错射变纯运气)。
- **⚠ 合并 = 连通块 flood-fill(2D 深度,仍完全确定可测)**——初版曾想「只竖向」,但那会退化成 5 条平行一维小游戏、没了跨列大连锁的爽点,故改连通块:
  - 弹药落定后进「结算循环直到稳定」:找所有**四邻相连、同值、≥2 的连通块** → 合成翻倍,新块落在该块**最低、再最左**的锚点格(确定),其余清空;
  - **重力向上贴顶**(远离死线=减压)压实 → 再找连通块 → 循环到无同值可合。
  - flood-fill + 固定锚点 + 固定重力方向 = 确定、可单测,但有了横+纵 2D 连锁。
- **⚠ A. 弹药值随进度放大**(防大鱼变「死墙」的核心平衡规则):弹药不恒为 2,而在**当前盘面较小的几档**里取(如 [最小档, 最小档×4] 区间),保证任何时候都合得动、不软锁。
- **⚠ E. 重力方向(纠正:统一「向上贴顶」)**:合并后由上述连通块结算统一向上压实。**空洞只有锤子会造**(砸中间块)→ 其**下方的块上浮贴顶**补位(远离死线=减压,是「上浮」不是「下沉」)。写进单测。
- **F. 连锁倍率**:一次射击连通块连锁合越多,倍率越高→分越高、金币越多、音效越爆(学 Roguebreaker 连击,给技巧表达与刷分深度)。

## 2. 压力与失败(⚠ C. 压力取向写死:错射为主、刷行为辅)

- **两个压力源**:① 你**射错**(不同数)→那列多一块、往下顶(自作自受,**主压力**);② 顶部**定期刷新行**下压(紧迫感,**给得温柔**)。
- 真正的技巧 = 少射错、多连锁。取向偏 Threes/2048 的纯技巧感,而不是被系统硬逼。刷行频率是关键平衡钮(蒙特卡洛调)。
- **唯一失败条件**:任意列被顶进**底部炮台行**(学 mines「一条干净规则」)。

## 3. 差异化钩子 —— 深海鱼图鉴(收集核心,**无水族馆,数字梯本身即图鉴**)

- **数字→鱼进化梯**:一套**精选 ~24 档标志鱼**(身份唯一清晰),`2→4→…→2048→…→~16M`,**浅礁 → 深渊**递进(越大数字=越深海生物,主题「深渊」由此而来)。合并动画 = 小鱼变大鱼。
  - 样例梯(P1 从 271 条里精选定表,均已确认存在):2 小丑鱼·4 鳚鱼·8 蝴蝶鱼·16 神仙鱼·32 河豚·64 梭鱼·128 黑鳍礁鲨·256 鮟鱇(深海起)·512 管眼鱼·1024 腔棘鱼·2048 大白鲨·4096 鲸鲨·8192 白鲸…更深无尽。
  - **⚠ 梯子顶端与巨数显示(无尽收尾)**:梯子有顶(~24 档);合到**最深鱼即封顶**——顶档鱼**不再合、当稳定块**(不做 prestige,保简)。巨数塞不进格子:**高档纯鱼图不显数字**(或缩写 16M/1B),中低档才显数。
- **图鉴 = 收集**:创到新最深的鱼 → 解锁其图鉴条目(参考游戏「512 Locked」正是下一条待解锁鱼)。DOM 浮层,抄 snake gallery。**这就是全部收集,不另设装饰馆**(用户拍板去掉水族馆)。
- **皮肤 = 纯背景/配色群系**(礁区/暮光/深渊/海沟),里程碑解锁,**只换水体背景与配色,不换鱼**(保数字身份清晰)。`render` 的 `PAL` 可切换引用,背景纹理函数必须确定性、禁 `Math.random`。

## 4. 轻 meta(这类型自带,别学 mines 硬堆系统)

- **金币 + 三道具**(参考截图):🏓 锤子(砸掉一块→下方块上浮贴顶)、🔀 **交换两列 或 清掉最底一行**(比抽象「洗牌」可控好测,列堆模型里 shuffle 语义不清故换掉)、↩ 撤销上一发;金币靠合并/连锁攒。
- **~120 成就**(学 snake 阶梯累计族 + 单局族,数据表驱动 + 运行时断言)。
- **最高分** + **每日挑战**(固定种子牌序,抄 mines 每日盘+streak)。
- **⚠ H. 撤销 vs 每日种子完整性**:撤销恢复完整前态**含 PRNG 游标**;**每日盘限用或禁用撤销**,防退回刷 RNG 作弊(同 snake AI 反刷精神)。
- **不做**:肉鸽选卡 / 永久数值升级树(核心拼图已够紧,规避 mines「系统过重失乐趣」教训)。

## 5. 广告(接引擎 `Ads`,每游戏自备 adUnits,绝不复用他游戏)

激励-复活(输了看广告清底部两行续命,每局限 2 次)· 激励-免费道具/金币 · 插屏(每 N 局间隙弹,非局内)。AI 代打不做(拼图类无意义)。

## 6. 工程(严守本仓引擎契约,见 `engine/README.md`;借 Abyss 架构概念)

- 走**本仓 `engine/`**(snake/mines 同款),**不 fork** Abyss 的 www/;借 Abyss 的**概念**:数字值→鱼图映射(单套梯,非 per-skin)、相位机、成就/i18n 表结构。
- **模块分层**(index.html 加载顺=依赖顺,load-bearing):
  引擎脚本 → `prng` → `tiles`(数字→鱼/梯数据) → `core`(网格/射击/连通块合并结算/弹药放大/向上重力/下压/失败) → `powerups`(锤子/交换列或清行/撤销) → `codex` → `storage` → `achievements` → `skins`(背景群系) → `render` → `main`。
- **纯逻辑层**(双导出 `if(module.exports)...else 全局`,node 可单测):`prng/tiles/core/powerups/achievements` + codex/skins 数据函数。**改这些先写/跑测试**。
- **DOM/渲染层**(E2E + 无头截图验):`render`(renderAll 契约、offscreen)、`main`(boot/循环/事件消费/存档/面板)、codex/皮肤 DOM。
- **游戏时钟非墙钟**:刷行节拍、动画时长走 `gameMs`;**core 禁 `Date.now()`**(每日种子豁免)。
- **可测性(整数网格,回到 snake/mines 水平)**:PRNG 牌序 + `s.events`(shoot/land/merge/chain/spawn/advance/hammer/death/newMaxFish…)统一供音效与成就消费。**node 单测**:连通块合并结算(锚点+向上重力+循环到稳定)正确、弹药放大区间、锤子上浮补位、连锁倍率、失败判定;**无头冒烟** + **E2E**(整局+道具+图鉴+皮肤+存档续玩+广告)。**⚠ 蒙特卡洛口径(无尽刷分无「可赢性」可验)**:验「无退化的秒死/永生局 + 局长分布合理 + 弹药区间不软锁」,调刷行频率与弹药区间——不是 mines 那种「必可赢」。
- **存档**(storage.js):单 JSON blob = 元进度(图鉴/解锁/成就/最高分/皮肤/金币/每日 streak) + 当局网格快照(列堆/弹药/下一发/分/gameMs)续玩。**`SAVE_VERSION` 纪律**:改形状必 bump、旧档丢弃不迁移;改 JS 必 bump `?v=N`;保守合并的**开放 map 字段保持空默认**(否则每次 load 清空动态 key,snake 实踩 Critical)。
- **零硬编码文案**:全走 `T('dotted.key')` + `locales/<lang>.json`(**必须嵌套结构**,扁平写法查不到且零报错——snake 实踩),en 基准 + zh-CN,商店页十语。canvas 非定长文案过 `wrapLines`。`node tools/check-locales.js` 必 0 fail。
- **iOS 壳**:自备 `package.json`(Cap6,抄 minesweeper)、`capacitor.config.json`、`resources/`;`codemagic.yaml` 加一段 workflow 填 4 var。
- 从 `games/_demo` 起步,新目录 `games/abyssshoot/`。

## 7. ⚠ G. 素材预处理(鱼不是拿来即用)

fishId 的 271 条是给「识鱼 App」做的,取景/姿态/尺寸不统一。当**格子图标**须过一道 pipeline:统一裁切 + 尺寸归一 + 投影,且**数字要能压在鱼上读清**(大数字 badge 叠鱼图,鱼当图标底)。走 `game-art-pipeline` skill;先 emoji/纯色占位开发,美术后补零改码。P1 从 271 条精选 ~24 条定梯 + 起 3~4 套背景群系。

## 8. 分阶段实施(学 mines 教训:先把核心拼图做好玩,别一上来堆系统)

1. **P1 核爽**:`core`(5 列 / 射击 / 连通块合并结算 / 弹药放大 / 向上重力 / 温柔刷行 / 唯一失败) + 精选一套鱼梯 + 连锁倍率 + node 单测 + 蒙特卡洛调平衡 + 无头冒烟 → **先确认「射鱼-连锁-被顶爆」手感成立**再往下。
2. **P2 meta**:金币 + 三道具(锤子/交换列或清行/撤销,含 H 撤销防作弊) + 图鉴 + 最高分 + 每日盘 + 事件流接音效。
3. **P3 收集/皮肤**:背景群系皮肤 + ~120 成就 + 存档续玩。
4. **P4 上架**:Ads 钩子 + i18n 补 + iOS 壳 + 商店素材 + 鱼素材精修。
